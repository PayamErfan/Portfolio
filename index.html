My Projects

NGordnet (Wordnet)

The project aims to incorporate WordNet, a semantic lexicon for the English language, into the NGordnet tool. The main task is to implement a HyponymsHandler that returns hyponyms of a given word based on the WordNet dataset. The project is structured in several steps, including creating a directed graph class to represent WordNet data, implementing hyponyms retrieval, handling lists of words, handling the k parameter (limiting the number of results), and optionally adding more features like plotting relative frequencies. The project emphasizes design decisions, graph operations, data processing, and interaction between Java, HTML, and JavaScript components.
